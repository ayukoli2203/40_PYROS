SmokeFree is more than a tracker or a quit‑buddy chatbot - it’s a unified framework that combines both, using behavioral science to help users overcome addiction

Core Features
- 4‑7‑8 Breathing Loop
Interactive timer with sound/visual cues (4s inhale, 7s hold, 8s exhale) to ease cravings in 3 minutes, with ocean‑wave background audio
- AI CBT Coach
Empathetic support using Cognitive Behavioral Therapy (CBT) to help reframe triggers.
- Vitality Gamification
An identity-based XP system where the vitality score (XP) increases as the user engages with tools and stays smoke-free.
- No‑Shame Recovery
Slip‑ups don’t reset everything — users keep 75% progress to encourage consistency.
- Interactive Insights
Real‑time charts of craving patterns, plus trackers for money saved and life regained.

Tech Stack
- Frontend: Pure HTML5, CSS3 (Keyframe Animations), and Vanilla JavaScript (ES6+).
- AI Engine: Anthropic Claude API (System-Prompted LLM).
- Storage: localStorage for offline-first data persistence.
- Aesthetics: Deep midnight-black theme with 4% opacity noise textures and teal-accented radial glows for focused immersion.

Impact & Social Relevance
- Accessibility: Optimized to run on low-spec (₹5,000) Android devices to support Tier 2-3 cities in India.
- Equity: Zero-cost, no-install PWA (Progressive Web App) architecture for immediate intervention.
- Scale: Targets the 267M tobacco users in India to mitigate the 1.35M annual death toll.
